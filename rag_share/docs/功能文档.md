# RAG 可视化演示 - 功能文档

## 概述

RAG 可视化演示是一个用于知识分享的 Web 网页，直观展示 RAG（检索增强生成）流程的各个步骤。页面以**手动分步控制**为核心，支持讲解者在任意步骤停留讲解。

---

## 主要功能

### 1. 手动步骤控制

页面顶部提供流程步骤导航栏，共 9 个步骤：

| 步骤 | 名称 | 说明 |
|------|------|------|
| 1 | 原始文本 | 展示输入的原始文档 |
| 2 | 预处理 | 文本清洗和标准化 |
| 3 | 切片 | 将文本切分成 chunks |
| 4 | 向量化 | 将 chunks 转为向量 |
| 5 | 用户提问 | 展示 Query 输入 |
| 6 | 相似度检索 | 计算相似度并召回 |
| 7 | 重排 | 对召回结果重排序 |
| 8 | Prompt 组装 | 组装最终 Prompt |
| 9 | 答案生成 | 展示最终回答 |

**控制按钮**：
- 上一步 / 下一步 - 逐步切换
- 点击步骤导航 - 跳转到指定步骤
- 重置 - 回到初始状态

### 2. 输入配置（左侧面板）

#### 原始文本输入
- 输入知识库文档内容
- 支持多段落文本

#### 用户问题输入
- 输入查询问题

#### 参数设置
| 参数 | 说明 | 默认值 |
|------|------|--------|
| Chunk Size | 每个 chunk 的字符数 | 50 |
| Overlap | 相邻 chunk 重叠字符数 | 10 |
| Top K | 召回的最相关 chunk 数量 | 3 |
| 切片方式 | 按字符 / 按句子 / 按段落 | 按字符 |
| 启用重排 | 是否启用重排序 | 开启 |
| PostgreSQL 混合检索 | 是否启用 PostgreSQL 两路召回 | 关闭 |

#### 一键填充
点击"填充示例内容"自动填入演示文本

### 3. 切片管理（步骤3 - 可编辑）

切片步骤支持手动编辑：

- **编辑** - 修改任意 chunk 的内容
- **新增** - 添加新切片
- **删除** - 移除切片
- **重置** - 恢复为原始切片

编辑后，后续步骤会自动使用编辑后的切片

### 4. 向量化展示（步骤4）

#### 自定义文本向量对比
- 输入两个文本
- 点击"计算相似度"
- 展示两个文本的向量热力图（前60维可视化）
- 显示余弦相似度值

#### 固定示例对比
预设三组示例：
- 语义相似
- 语义不同
- 完全无关

点击"获取向量"获取真实 embedding 并展示

#### 完整相似度矩阵
- 表格展示所有 chunks 之间及与 query 的相似度
- 色块颜色表示相似度高低

#### 向量详情
可折叠展开查看每个 chunk 的向量详情

### 5. 检索结果展示（步骤6）

#### 检索模式
- **内存模式**（默认）：使用本地向量数据库进行语义检索
- **PostgreSQL 混合检索**：启用两路召回 + 分数融合

#### PostgreSQL 混合检索
启用后，系统执行三阶段检索：

1. **第一路 - 向量语义检索**
   - 使用 pgvector HNSW 索引
   - 基于语义相似度召回相关 chunk

2. **第二路 - 全文关键词检索**
   - 使用 PostgreSQL ILIKE 模式匹配
   - 提取查询中的中文 bigram 和英文关键词
   - 统计每个 chunk 匹配到的关键词数量

3. **融合结果**
   - 融合公式：`融合分 = 0.6 × 向量分 + 0.4 × 全文分`
   - 按融合分排序返回 Top-K

#### 分数说明
| 分数类型 | 说明 | 范围 |
|---------|------|------|
| 向量分 | 向量语义相似度 | 0-1 |
| 全文分 | 关键词匹配率（匹配数/总关键词数） | 0-1 |
| 融合分 | 加权融合分数 | 0-1 |

#### Query 向量
可折叠展示，点击展开查看完整 2048 维向量

#### 召回结果
- 按融合分排序
- Top-K 高亮显示
- 未命中项置灰
- 混合检索模式下显示三个分数条（向量/全文/融合）

### 6. 重排对比（步骤7）

- 左侧：初始召回排序
- 右侧：重排后排序
- 排名变化箭头标注

### 7. Prompt 组装（步骤8）

代码块展示完整 Prompt：
- System Prompt（系统指令）
- Context（检索到的上下文）
- User Query（用户问题）

### 8. 答案生成（步骤9）

- 展示最终回答
- 附带引用了哪些 chunk

## 使用场景

- 技术分享现场演示
- 内部培训
- 录屏讲解
- 向非技术同学解释 RAG 原理

---

## 操作流程

1. 在左侧输入原始文本和查询问题
2. 调整参数（可选）
3. 点击"开始演示"
4. 点击"下一步"逐步推进
5. 在任意步骤停留讲解
6. 可随时点击步骤导航跳转
7. 切片步骤可手动编辑内容
8. 向量化步骤可输入自定义文本测试相似度
