# 启动指南

## 方式一：使用虚拟环境（推荐）

```bash
# 进入项目目录
cd /Users/xuzishuo/ai-work/狼人杀/backend

# 激活虚拟环境
source venv/bin/activate

# 启动服务器
python app.py
```

## 方式二：使用启动脚本

```bash
cd /Users/xuzishuo/ai-work/狼人杀/backend
./run.sh
```

## 访问游戏

打开浏览器，访问：http://localhost:5000

## 配置GLM模型

1. 访问 [智谱AI](https://open.bigmodel.cn/) 获取API Key
2. 创建 `.env` 文件：
   ```bash
   cp .env.glm .env
   ```
3. 编辑 `.env` 文件，填入API密钥：
   ```env
   LLM_PROVIDER=glm
   GLM_API_KEY=你的API密钥
   DEFAULT_MODEL=glm-4-plus
   ```
4. 在游戏界面选择 **GLM-4/5** 作为LLM提供商

## 游戏操作

1. 选择玩家数量（6-12人）
2. 选择LLM提供商（模拟AI / GLM-4/5 / OpenAI / Anthropic）
3. 点击"开始游戏"
4. 观看AI自主博弈
5. 查看右侧面板的AI思考过程

## 常用命令

```bash
# 激活虚拟环境
source venv/bin/activate

# 停止服务器
按 Ctrl+C

# 退出虚拟环境
deactivate
```

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| 端口被占用 | 修改 `app.py` 中的端口号 |
| 依赖安装失败 | 确保在虚拟环境中运行 |
| 模块导入错误 | 确保已激活虚拟环境 |
| GLM API调用失败 | 检查API密钥和网络连接 |
