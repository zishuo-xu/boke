# 快速启动指南

## 最快启动方式（3步）

### 1. 进入项目目录
```bash
cd /Users/xuzishuo/ai-work/狼人杀/backend
```

### 2. 安装依赖（首次运行）
```bash
pip3 install -r requirements.txt
```

### 3. 启动服务器
```bash
python3 app.py
```

## 访问游戏

打开浏览器，访问：**http://localhost:5000**

## 常用命令

```bash
# 使用启动脚本（自动安装依赖）
./run.sh

# 手动启动
python3 app.py

# 停止服务器
按 Ctrl+C
```

## 游戏操作

1. **选择人数**：6-12人局
2. **选择LLM**：模拟AI / GLM-4/5 / OpenAI / Anthropic
3. **点击"开始游戏"**
4. **观看AI自主博弈**
5. **查看思考过程**：右侧面板实时显示
6. **控制游戏**：暂停/继续/重置/调节速度

## 配置GLM模型（推荐）

1. 访问 [智谱AI](https://open.bigmodel.cn/) 获取API Key
2. 创建 `.env` 文件：

```env
LLM_PROVIDER=glm
GLM_API_KEY=你的API密钥
DEFAULT_MODEL=glm-4-plus
```

3. 在游戏界面选择 **GLM-4/5** 作为LLM提供商

> 也可以使用OpenAI或Anthropic，或选择"模拟AI"快速体验

详细配置请查看 [GLM_SETUP.md](GLM_SETUP.md)

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| 端口被占用 | 修改 `app.py` 中的端口号 |
| 依赖安装失败 | 升级 pip: `pip3 install --upgrade pip` |
| 页面无法访问 | 检查防火墙设置 |
