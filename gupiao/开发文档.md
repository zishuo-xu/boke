# 股票/基金数据监控面板 - 开发文档

## 项目概述

**项目名称**: 股票/基金数据监控+AI解读面板

**开发时间**: 2026年2月

**技术栈**:
- 后端: Python 3.14 + FastAPI + SQLAlchemy + AKShare
- 前端: 原生 JavaScript + ECharts
- 数据库: SQLite
- 数据源: AKShare（开源财经数据接口）

**核心功能**:
1. 用户认证系统（注册/登录/登出）
2. 自选管理（股票/基金增删改查、分组管理）
3. 数据监控（实时行情、历史走势、图表展示）
4. 价格预警（上下限预警、通知提醒）
5. AI解读（预留接口）

## 数据库表结构

### 用户表 (users)
```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    email VARCHAR(100) NOT NULL UNIQUE,
    password_hash VARCHAR(255) NOT NULL,
    nickname VARCHAR(50),
    created_at TIMESTAMP NOT NULL,
    updated_at TIMESTAMP NOT NULL
);
```

### 分组表 (groups)
```sql
CREATE TABLE groups (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    name VARCHAR(50) NOT NULL,
    sort_order INTEGER DEFAULT 0,
    created_at TIMESTAMP NOT NULL,
    updated_at TIMESTAMP NOT NULL,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);
```

### 标的表 (stocks)
```sql
CREATE TABLE stocks (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    code VARCHAR(20) NOT NULL,
    name VARCHAR(100) NOT NULL,
    type VARCHAR(10) NOT NULL CHECK (type IN ('stock', 'fund')),
    market VARCHAR(10),
    sector VARCHAR(50),
    fund_company VARCHAR(100),
    fund_type VARCHAR(50),
    group_id INTEGER,
    sort_order INTEGER DEFAULT 0,
    created_at TIMESTAMP NOT NULL,
    updated_at TIMESTAMP NOT NULL,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (group_id) REFERENCES groups(id) ON DELETE SET NULL
);
```

### 预警表 (alerts)
```sql
CREATE TABLE alerts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    stock_id INTEGER NOT NULL,
    alert_type VARCHAR(10) NOT NULL CHECK (alert_type IN ('upper', 'lower')),
    threshold DECIMAL(10, 4) NOT NULL,
    enabled BOOLEAN DEFAULT TRUE,
    triggered_at TIMESTAMP NULL,
    created_at TIMESTAMP NOT NULL,
    updated_at TIMESTAMP NOT NULL,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (stock_id) REFERENCES stocks(id) ON DELETE CASCADE
);
```

### 预警通知表 (alert_notifications)
```sql
CREATE TABLE alert_notifications (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    alert_id INTEGER NOT NULL,
    message VARCHAR(500) NOT NULL,
    is_read BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP NOT NULL,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (alert_id) REFERENCES alerts(id) ON DELETE CASCADE
);
```

## 开发过程中解决的问题

### 问题 1: bcrypt 密码哈希 72 字节限制
**错误**: `password cannot be longer than 72 bytes`

**原因**: bcrypt 算法限制密码长度为 72 字节

**解决方案**:
1. 更改密码哈希方案为 Argon2（`schemes=["argon2"]`）
2. Argon2 无长度限制，更现代更安全

**文件**: `app/core/security.py`

### 问题 2: Pydantic v2 兼容性
**错误**: `ResponseValidationError: 2 validation errors: created_at/updated_at`

**原因**: UserResponse 中 `created_at` 和 `updated_at` 字段类型为 `Optional[str]`，但数据库返回的是 `datetime` 对象

**解决方案**:
1. 将字段类型改为 `Optional[datetime]`
2. 在 schemas 中导入 `from datetime import datetime`

**文件**: `app/schemas/user.py`

### 问题 3: JWT Token 类型转换
**错误**: `HTTPException(detail="Not authenticated")`

**原因**: JWT payload 中的 `sub` 字段为字符串，但代码期望为整数

**解决方案**:
```python
# 添加类型转换
try:
    user_id = int(payload.get("sub"))
except (ValueError, TypeError):
    raise HTTPException(...)
```

**文件**: `app/core/dependencies.py`

### 问题 4: 页面跳转问题
**错误**: 登录成功后跳转回登录页

**原因**: app.html 加载了 auth.js，导致重复的认证检查

**解决方案**:
- 从 app.html 中移除 auth.js 引用
- auth.js 只在 index.html 中使用

**文件**: `frontend/app.html`

### 问题 5: 浏览器缓存问题
**错误**: 修改后前端仍显示旧版本

**解决方案**:
- 为 HTML 响应添加缓存控制头
```python
headers={"Cache-Control": "no-cache, no-store, must-revalidate"}
```

**文件**: `app/main.py`

## 新增功能：股票代码查询

### 需求
无需登录即可根据股票代码查询实时行情

### 实现步骤

1. **后端服务层** (`app/services/market_service.py`)
```python
def get_stock_by_code(self, stock_code: str, force_refresh: bool = False) -> Optional[Dict]:
    cache_key = f"stock:code:{stock_code}"
    if not force_refresh:
        cached_data = cache_service.get(cache_key)
        if cached_data:
            return cached_data
    data = self.data_fetcher.get_stock_realtime_data(stock_code)
    if data:
        cache_service.set(cache_key, data, self.stock_cache_ttl)
    return data
```

2. **API 路由** (`app/api/v1/market.py`)
```python
@router.get("/stock/code/{stock_code}", response_model=StockRealtimeData)
def get_stock_by_code(stock_code: str, force: bool = Query(False)):
    data = market_service.get_stock_by_code(stock_code, force)
    return data
```

3. **前端调用** (`frontend/static/js/api.js`)
```javascript
const marketAPI = {
    getStockByCode: (code, force = false) => {
        return apiRequest(`/market/stock/code/${code}?force=${force}`);
    },
    // ...
};
```

4. **前端界面** (`frontend/app.html`)
```html
<div class="search-box">

```

5. **前端逻辑** (`frontend/static/js/app.js`)
