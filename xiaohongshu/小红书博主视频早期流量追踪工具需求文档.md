# 2. 核心功能需求

## 2.1 博主管理（动态指定博主）

- 新增博主：支持手动录入博主基础信息（博主昵称、小红书账号链接、备注，可选），完成博主添加。

- 编辑/删除博主：支持修改已添加博主的信息，可删除无需继续追踪的博主，删除前弹出确认提示，避免误操作。

- 博主切换：顶部提供博主下拉选择器，一键切换不同博主，切换后工具内所有数据同步刷新为当前选中博主的相关内容，不同博主数据严格隔离，互不干扰。

## 2.2 视频管理（动态指定视频）

- 新增视频：选中目标博主后，可添加该博主的视频，录入视频基础信息（视频标题、视频链接、发布时间、分类/标签，标签可选）。

- 视频筛选：支持按博主筛选（默认显示当前选中博主的所有视频），可按时间段（自定义起止时间）筛选，也可按流量表现（全部、爆款、低流量，可自定义爆款/低流量判断阈值）筛选。

- 视频搜索：支持通过视频标题、视频链接关键词搜索，快速定位单条视频，方便查看具体数据。

- 视频编辑/删除：支持修改视频基础信息，可删除无需继续追踪的视频，删除后该视频所有数据同步删除。

## 2.3 数据录入功能

- 分段录入：每条视频支持按时间节点分段录入数据，无需一次性完成，可后续补录、修改数据。

- 强制录入节点（必录）：
        

    - 发布后1小时：播放量、点赞数、收藏数、评论数、完播率（数据需与小红书后台一致）；

    - 发布后4小时：播放量、点赞数、收藏数、评论数、完播率；

    - 发布后8小时：播放量、点赞数、收藏数、评论数、完播率。

- 可选录入节点：发布后24小时，可手动补充播放量、点赞数、收藏数、评论数、完播率，补充后自动融入流量曲线。

- 录入提示：录入数据时，标注数据单位（播放量无单位、点赞/收藏/评论数无单位、完播率为百分比），避免录入错误。

## 2.4 数据展示功能（核心）

### 2.4.1 单视频数据展示

- 数据卡片：清晰展示该视频的基础信息（博主、标题、发布时间）及各时间节点的完整数据（1h、4h、8h必显，24h如有则显），数据排版简洁，一目了然。

- 流量曲线：自动生成折线图，X轴为时间节点（1h→4h→8h→24h），Y轴支持切换展示播放量、点赞数、收藏数、评论数、完播率，可单独查看单个指标曲线，也可叠加查看多个指标曲线，直观呈现早期流量增长速度和变化趋势。

### 2.4.2 视频列表数据展示

视频列表默认显示当前选中博主的所有视频，每条视频仅展示核心信息：视频标题、发布时间、1h播放量、8h播放量，点击可进入单视频详情页查看完整数据和流量曲线。

## 2.5 数据导出功能

- 导出范围：支持导出当前选中博主的所有视频数据，也可选中单条/多条视频导出对应数据。

- 导出格式：支持Excel、CSV两种格式，导出内容包含视频基础信息及1h、4h、8h、24h（如有）的全部数据，适配博主做月报、数据汇总或对接品牌方。

## 2.6 界面与交互需求

- 极简界面：聚焦核心功能，无多余广告和冗余模块，主要分为三个页面：博主选择+视频列表页、视频数据录入页、单视频数据详情页。

- 交互便捷：操作步骤简单，数据录入、博主切换、视频筛选、导出等功能均不超过3步即可完成。

- 响应式适配：支持PC端、移动端（手机），界面布局自动调整，确保移动端查看数据、录入数据流畅。

# 3. 数据规范

## 3.1 数据字段规范

|数据类型|字段名称|数据要求|是否必录|
|---|---|---|---|
|博主信息|博主昵称|不超过20字，与小红书账号一致|是|
|博主信息|小红书账号链接|有效链接，可直接跳转|否|
|视频信息|视频标题|不超过50字，与小红书视频标题一致|是|
|视频信息|视频链接|有效链接，可直接跳转小红书视频|是|
|视频信息|发布时间|精确到分钟，与小红书发布时间一致|是|
|流量数据|播放量/点赞数/收藏数/评论数|非负整数，与小红书后台数据一致|是（1h/4h/8h）|
|流量数据|完播率|0-100之间的数字（可保留1位小数），百分比形式|是（1h/4h/8h）|
## 3.2 数据隔离规范

不同博主的视频数据、流量数据严格隔离，切换博主后，仅显示当前博主的相关内容，不出现其他博主的数据，确保数据不混淆。
